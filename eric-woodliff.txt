1. What is Semantic HTML? 
Semantic HTML, comprised of elements such as header, nav, footer, section, summary, and many more, describes the purpose of an element so that both machines and people can understand the content of a webpage. 
2. What is HTML used for? 
HTML is a markup language used to structure a webpage. 
3. What is an attribute and where do we put it? 
Attributes give us additional information about elements. We put attributes in the start tag.
4. What is the h1 tag used for? How many times should I use it on a page?
The h1 tag is used for the heading of a webpage. I should only use it once on a page.

6. What do we put in the head of our HTML document? 
The title must go in the head of our HTML document. Scripts, metadata, and style sheets can be applied here also. 
7. What is an id? 
id is used to establish a unique id for an element. An id should only be used once. 
8. What elements can I add an id to? 
HTML elements such as paragraphs and headings. 
9.How many times can I use the same id on a page?

10. What is a class? 
An element selector that can be used on any HTML element. It is generally paired with CSS for styling an HTML element and you can have as many elements as you desire in a class. 
11. What elements can I add a class to? 
Any HTML element. 
12. How many times can I use the same class on a page? 
As many times as I want.
